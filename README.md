**Key Features:**
- 🔐 JWT + Google OAuth authentication with 3 roles (User, Artist, Admin)
- 🖼️ Artwork discovery — hero carousel, search, filters, pagination
- 💳 Stripe Checkout — one-time purchases + subscription tiers (Free/Pro/Premium)
- 💬 Purchase-gated comment system with edit/delete
- 📊 Dashboards — User (purchases, wishlist), Artist (CRUD artworks, sales), Admin (analytics, charts)
- 🎨 Dark mode, skeleton loaders, Framer Motion animations, responsive design
- 📧 Wishlist, email notifications, imgBB image upload
